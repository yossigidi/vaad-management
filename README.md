# ניהול ועד בית

מערכת לניהול ועד בית - גביית דמי ועד, מעקב הוצאות, ניהול פרויקטים ודיירים.

## תכונות

- 📊 **דשבורד** - מבט מהיר על מצב הגביה, חייבים והיתרה
- 👥 **ניהול דיירים** - 16 דירות עם פרטי קשר ושליחת תזכורות בוואטסאפ
- 💰 **תשלומי ועד** - מעקב חודשי, חייבים מסומנים באדום
- 🧾 **הוצאות** - קבועות ומשתנות, חלוקה לקטגוריות
- 🔨 **פרויקטים** - יצירת פרויקטים עם חלוקה אוטומטית בין הדיירים, הוצאות פר פרויקט
- 📈 **דוחות** - ייצוא לאקסל, הדפסה, גיבוי JSON

## הרצה מקומית

```bash
npm install
npm run dev
```

## בילד

```bash
npm run build
```

## טכנולוגיות

- React 18 + Vite
- Tailwind CSS
- localStorage (כל הנתונים נשמרים בדפדפן)
- Cloudflare Pages לדיפלוי
